# Tox
#### Установка зависимостей
```
pip install -r requirements.txt
```
#### Запуск tox
```
tox
```

# Git
#### Слияние hey в main
```
git checkout main
git merge hey
```
#### Подтянуть изменения из main в hey
```
git checkout hey
git rebase main
```
#### Отменить изменения в файле requirements.txt
```
git checkout requirements.txt
```
